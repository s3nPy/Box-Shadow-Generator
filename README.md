## Overview

Simple application that helps you adjust box-shadow css property visually in real time 

## Build and start

##### 1. Install dependencies
```Shell
$ npm install
```

##### 2. Start application (dev mode)
```Shell
$ npm start
```

###### 3. Build application (optional)
```Shell
$ npm run build
```
